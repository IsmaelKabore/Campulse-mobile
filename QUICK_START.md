# Quick Start - Android Studio

## üöÄ Step-by-Step Guide

### 1. Open Project in Android Studio

1. **Launch Android Studio**

2. **Open the Flutter Project:**
   - Click **"Open"** on the welcome screen
   - Or: **File** ‚Üí **Open**
   - Navigate to: `C:\Users\ismae\Documents\GitHub\campulse-mobile`
   - **Important:** Select the **root folder** (`campulse-mobile`), NOT the `android` folder
   - Click **"OK"**

3. **Android Studio will detect it's a Flutter project**
   - If prompted to install Flutter/Dart plugins, click **"Yes"**
   - Wait for indexing to complete

### 2. Install Flutter Plugin (if needed)

1. **File** ‚Üí **Settings** (or press `Ctrl + Alt + S`)
2. Go to **Plugins**
3. Search for **"Flutter"** and click **"Install"** (Dart will install automatically)
4. Click **"Apply"** and restart Android Studio if prompted

### 3. Configure Flutter SDK

1. **File** ‚Üí **Settings** ‚Üí **Languages & Frameworks** ‚Üí **Flutter**
2. Click the folder icon next to **"Flutter SDK path"**
3. Navigate to where Flutter is installed:
   - Common locations: `C:\src\flutter` or `C:\flutter`
   - Or find it by running `where flutter` in Command Prompt
4. Click **"Apply"** ‚Üí **"OK"**

### 4. Sync Gradle

1. Android Studio should show a notification: **"Gradle files have changed"**
2. Click **"Sync Now"**
   - Or manually: **File** ‚Üí **Sync Project with Gradle Files**
3. Wait for sync to complete (check bottom status bar)

### 5. Create/Start Android Emulator

#### Option A: Create New Emulator

1. Click the **Device Manager** icon (üì± phone icon) in the toolbar
   - Or: **Tools** ‚Üí **Device Manager**

2. Click **"+ Create Device"**

3. **Select Hardware:**
   - Choose **"Pixel 5"** or **"Pixel 6"** (recommended)
   - Click **"Next"**

4. **Select System Image:**
   - Choose **"Tiramisu" (API 33)** or **"UpsideDownCake" (API 34)**
   - If not downloaded, click **"Download"** next to it
   - Wait for download, then click **"Next"**

5. **Verify Configuration:**
   - Name: `Pixel_5_API_33` (or your choice)
   - Click **"Finish"**

#### Option B: Use Existing Emulator

1. Open **Device Manager** (üì± icon)
2. Find your emulator in the list
3. Click the **Play button (‚ñ∂)** to start it
4. Wait 1-2 minutes for it to boot

### 6. Run the App

#### Method 1: Using Android Studio UI

1. **Select your emulator** from the device dropdown (top toolbar)
   - Should show: `Pixel_5_API_33` or your emulator name

2. Open `lib/main.dart` in the editor

3. Click the **Run button** (‚ñ∂ green play icon) in the toolbar
   - Or press `Shift + F10`
   - Or right-click `main.dart` ‚Üí **Run 'main.dart'**

#### Method 2: Using Terminal

1. Open the **Terminal** tab at the bottom of Android Studio

2. Check available devices:
   ```bash
   flutter devices
   ```
   (Your emulator should be listed)

3. Run the app:
   ```bash
   flutter run
   ```

### 7. First Run Setup

On first run, you may need to:

1. **Accept Android licenses:**
   ```bash
   flutter doctor --android-licenses
   ```
   (Type `y` for each license)

2. **Get Flutter dependencies:**
   ```bash
   flutter pub get
   ```

3. **Configure Firebase** (if not done):
   ```bash
   dart pub global activate flutterfire_cli
   flutterfire configure
   ```

## üêõ Common Issues & Fixes

### "Flutter SDK not found"
- Set Flutter SDK path in: **File** ‚Üí **Settings** ‚Üí **Flutter**

### "Gradle sync failed"
- Check internet connection
- Try: **File** ‚Üí **Invalidate Caches** ‚Üí **Invalidate and Restart**
- Or run in terminal: `cd android && ./gradlew clean`

### "No devices found"
- Start emulator first from Device Manager
- Or connect a physical device via USB

### "local.properties not found"
- This file is auto-generated by Android Studio
- If missing, create `android/local.properties` with:
  ```
  sdk.dir=C\:\\Users\\ismae\\AppData\\Local\\Android\\Sdk
  flutter.sdk=C\:\\path\\to\\flutter
  ```

### Emulator won't start
- Enable **Virtualization** in BIOS
- Check **Hyper-V** is enabled (Windows)
- Try increasing emulator RAM in AVD settings
